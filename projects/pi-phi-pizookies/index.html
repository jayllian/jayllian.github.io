Pi Phi Pizookies

About the Project

Pi Phi Pizookies is a recap video of the UC Davis’ Pi Beta Phi’s annual philanthropic fundraiser, Pi Phi Pizookie. This recap highlighted the behind-the-scenes work by the sisters of Pi Beta Phi to host this large, annual event.
